Ejyle.DevAccelerate.Core.Logging Namespace

DevAccelerate Reference Home
The Ejyle.DevAccelerate.Core.Logging namespace contains types that define basic logging mechanism for DevAccelerate. It also contains a default log management implementation.
Classes
  ClassDescription
DaDefaultLogManager
Provides default implementation of IDaLogManager interface.
DaLogManagerFactory
Represents the factory for creating IDaLogManager.
DaLogMessage
Represents the default implementation of IDaLogMessage interface.
Interfaces
  InterfaceDescription
IDaLogManager
Defines the methods for writing messages to a log.
IDaLogMessage
Represents a log message.
