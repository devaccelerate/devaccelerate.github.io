Ejyle.DevAccelerate.Core Namespace

DevAccelerate Reference Home
The Ejyle.DevAccelerate.Core namespace contains types for entity management, and global application context.
Classes
  ClassDescription
DaAsyncHelper
A helper class to call asynchronous methods synchronously.
DaEntityBase
Represents the base class for an entity. The type of the entity ID is Integer.
DaEntityBaseTKey
Represents the base class for an entity.
DaEntityManagerBaseTKey, TEntity
Represents the base class for entity managers with common properties and methods.
DaEntityResult
Represents the result of an operation on an entity or list of entities.
DaError
Represents an error related for an entity result.
DaGlobalApplicationContext
Represents the APIs to manage the global context of a DevAccelerate application. This class cannot be inherited.
